## 需注意的邊界案例

### 取消 vs キャンセル

- **取り消し**（操作の取り消し）→ 「撤銷操作」語意，用於 undo 場景
- **キャンセル**（カタカナ） → Dialog 的「取消」按鈕，Goodpatch 設計慣例建議用カタカナ
- 兩者都是正確日文，但語意不同。Futari 的 Dialog cancel button 確認用哪個後記錄在此。

### 精算 vs 清算

- **精算**（せいさん）→ 費用結算、割り勘精算 ✅ Futari 正確用法
- **清算**（せいさん）→ 公司廢業、債務清算 ❌ 家計簿情境錯用

### 娯楽 vs エンタメ

- 兩者都見於實際 App，娯楽略正式，エンタメ較口語。Futari 選擇使用漢字「娯楽」→ 白名單保留。

## 更新指引

新增詞彙時，請：

1. 確認該詞在日本家計簿 App 實際 UI 或財務文章中以漢字書寫（搜尋「夫婦 家計簿 [詞彙]」「家計管理 [詞彙]」等）
2. 填入漢字、読み、用途說明、確認來源
3. 若有邊界案例（同音異義、漢字/假名並存），加入「需注意的邊界案例」區段
4. 更新文件頂部的「最後更新」日期
