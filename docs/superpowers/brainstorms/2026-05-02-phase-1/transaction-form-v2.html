需要你確認的點

1. 欄位順序
金額 → 誰付的 → 描述 → 日期 → 分類 → 分攤方式。是這個順序嗎？
2. 金額擺最上 (hero)
數字鍵盤 auto-pop、佔最大視覺。OK 嗎？
3. 日期
預設今天、可改（補記用）。要不要列出來？或藏在「進階」？
4. 分類仍可選
第 3 題決定 category 可空。要不要在 chip 列加一顆「未分類」？或不選就 = 未分類？
5. 平分卡的 hint
「對方欠你 NT$ 120」即時計算 — 可以幫使用者確認金額對不對。OK 嗎？
6. 送出按鈕
放在 header 右上（iOS 慣例）。或要底部固定 bar？
